@Introduction:

This program implement a simple library management system.

***********************************************************************************
@Environment:

java version "9.0.1"
Java(TM) SE Runtime Environment (build 9.0.1+11)
Java HotSpot(TM) 64-Bit Server VM (build 9.0.1+11, mixed mode)

mysql  Ver 8.0.11 for osx10.13 on x86_64

JDBC mysql-connector-java-5.1.46-bin.jar

***********************************************************************************

Install application:

1.Install the MySQL on the user computer.
2.upload "library.sql" into user databases.
3.Use IDE to import the JAVA source code and plug-in the JDBC driver into external libraries of the JAVA project. Setup the path.
4.Run the program.
